# Features

## Core Calculator
- Display panel showing current input and running result
- Support for basic operations: addition, subtraction, multiplication, division
- Decimal point input support
- Clear (C) button to reset the calculator state
- Backspace (⌫) button to delete the last entered digit
- Equals (=) button to evaluate the expression
- Keyboard support (number keys, operators, Enter, Backspace, Escape)

## UI / UX
- Minimal single-page layout — calculator centered on a neutral background
- Responsive design using Tailwind CSS utility classes
- Subtle hover and active states on buttons for tactile feedback
- Error display for invalid operations (e.g. division by zero)
- No login, no database, no network calls — fully offline-capable
